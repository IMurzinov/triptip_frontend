# triptip_frontend

TripTip — веб-фронтенд для туристического приложения. Платформа позволяет пользователям создавать, просматривать и делиться своими путешествиями.

**Backend репозиторий:** https://github.com/cosmofactory/triptip_backend

## Установка зависимостей

```sh
npm install
```

## Сборка проекта

```sh
npm run build
```

## Запуск локального сервера

```sh
npm start
```

После запуска приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000).

## Дополнительные команды

### Запуск тестов

```sh
npm test
```

### Проверка синтаксиса

```sh
npm run build
```

## Технологии

- **React** 18.3.0 - основной фреймворк
- **Redux Toolkit** - управление состоянием
- **React Router** - роутинг
- **React Hook Form** - работа с формами
- **React Dropzone** - загрузка файлов
- **React Beautiful DnD** - drag-and-drop интерфейс

## Структура проекта

```
src/
├── components/     # Переиспользуемые компоненты
├── pages/          # Компоненты страниц
├── views/          # Представления и формы
├── api/            # API клиент и запросы
├── features/       # Redux слайсы
├── constants/      # Константы приложения
└── styles/         # Глобальные стили
```

## Разработка

Проект использует Create React App с настройками по умолчанию. Для разработки запустите:

```sh
npm start
```

Приложение автоматически перезагружается при изменении файлов.
